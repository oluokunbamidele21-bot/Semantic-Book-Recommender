📚 Semantic Book Recommender (LLM Powered)

🚀 Overview

This project is an AI-powered Semantic Book Recommendation System that leverages the power of Large Language Models (LLMs) and vector search to deliver intelligent and context-aware book suggestions.

Unlike traditional recommenders, this system understands the meaning behind user queries to return highly relevant books.

⸻

🧠 Key Features
	•	🔍 Semantic search (not just keyword matching)
	•	🤖 Powered by LLMs
	•	📊 Vector database for fast retrieval
	•	🎯 Personalized recommendations based on user intent
	•	💡 Interactive and modern UI

⸻

🛠️ Tech Stack
	•	Python
	•	LLMs (via Hugging Face)
	•	LangChain
	•	Chroma (Vector Database)
	•	Gradio / NiceGUI (Frontend)

⸻

⚙️ How It Works
	1.	Books are split into chunks using text splitters
	2.	Each chunk is converted into embeddings
	3.	Embeddings are stored in a vector database
	4.	User query is embedded and compared
	5.	Most relevant books are retrieved and displayed
